About

I was searching around the Internet for the better part of a day trying to find the necessary SDKs and compiler to put some custom software on a HP Jornada 540. This repository contains the Microsoft(TM) eMbedded Visual Tools, consisting of a C++ and Visual Basic compiler for the SH3 processor, among others.
